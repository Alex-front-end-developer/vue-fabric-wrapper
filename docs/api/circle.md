# Circle Component
[Official Docs Link](http://fabricjs.com/docs/fabric.Circle.html)


## Example Usage
```html
<fabric-canvas>
	<fabric-circle :id="'someId'" :radius="50"></fabric-circle>
</fabric-canvas>
```

## props
| Name | Type               | Required | Default |
| ---- | ------------------ | -------- | ------- |
| id | `Number`/`String` | `yes` | |
| radius| `Number` | `no` | 50 |
